Node Beam Overview

BeamNG vehicles are comprised of mainly two components, nodes and beams.

Nodes

A node is like a joint in a skeleton. Beams are attached to the node and can rotate around them. Nodes define the weight of the vehicle.

Beams

Beams are like bones within a skeleton. Between the nodes one has to put beams in order to make a physical structure (skeleton) of the vehicle.
